INTERROGATING BLACKS — Standalone Build
========================================

To run:
  1. Keep `InterrogatingBlacks.exe` and `better_sqlite3.node` in the same folder.
  2. Double-click InterrogatingBlacks.exe (or run it from a terminal).
  3. Open http://localhost:3847 in your browser.

Files created on first run (in this folder):
  - interrogating.db   (SQLite database, all stats/achievements live here)
  - uploads/           (custom question media)

Stack: Node.js 18 + Express + Socket.io + SQLite + React (prebuilt).
